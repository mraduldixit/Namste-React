2.  what is CDN ->

contant delievery network . CDN stands for Content Delivery Network. It is a distributed network of servers strategically located

  3.what is crossorigin ->

  The crossorigin attribute is an HTML attribute that can be used with the <script> tag to specify how the browser
   should handle loading and executing scripts from external domains or origins. It is primarily used to control how
    the browser enforces cross-origin resource sharing (CORS) policies for script files.
